## Troubleshooting

### Issue: Can't decide page type

**Solution:**
- Review validated intent—page type should match intent
- Consider what content types rank for similar keywords
- Review Chapter 1 for page type guidelines
- Remember: Informational → Articles, Commercial → Comparison pages, Transactional → Product pages

### Issue: Keywords seem to overlap

**Solution:**
- Group very similar keywords on the same page
- Use one primary keyword per page
- Create category pages for related keywords
- Ensure each page has a distinct purpose

### Issue: Can't create site structure

**Solution:**
- Start with homepage
- Create main category pages
- Add individual pages under categories
- Keep it simple—structure should be logical and navigable
