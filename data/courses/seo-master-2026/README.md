# SEO Master Course 2026

## Course Structure

This folder contains all data for the SEO Master Course 2026.

## File Organization

- `course.json` - Course metadata (title, description, instructor, etc.)
- `structure.js` - Course structure (days, chapters, labs)
- `tools.json` - Tools registry for this course
- `content/` - Course content files
  - `chapters/` - Chapter markdown files (books)
  - `labs/` - Lab markdown files
- `assets/` - Course assets
  - `visuals/` - Visual assets (SVGs, images)

## Adding Content

1. Add chapter files to `content/chapters/`
2. Add lab files to `content/labs/`
3. Update `structure.js` with new content references
4. Add visual assets to `assets/visuals/` as needed
5. Update `tools.json` if new tools are introduced
